Docs and hosting

Ship Lightning Studio as a static app

This build is pure HTML, CSS, and JavaScript, so it can be hosted on GitHub Pages the same way as the rest of your static tools.

Folder structure

`tools/lightning-studio/` contains the landing page, docs, tool directory, shared assets, and dedicated tool routes.

GitHub Pages

Push the repo, publish the branch, and the app works without any build step. Each tool route is a plain `index.html` page.

Privacy model

Formatting, generation, diffing, and log analysis run in the browser. REST and GraphQL explorers send requests directly from the client, so CORS rules still apply.

Suggested deploy paths

Use `https://your-domain/tools/lightning-studio/` when integrating into your existing site. The shared navigation already links back to the main tool directory and docs page.
